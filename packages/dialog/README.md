## Basic usage

This is a basic example how this component is used

```svelte
<script lang="ts">
  import { Dialog } from "@skui/dialog";
</script>

<Dialog>
  <h1>Hello World</h1>
  <p>This is a dialog.</p>
</Dialog>
```
